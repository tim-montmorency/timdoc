 2-Rapport de performance

Méthodes d'évaluation de la vitesse de votre site web

Avant de commencer l'optimisation de la vitesse de votre site web, vous devez déterminer votre temps de chargement actuel et définir ce qui ralentit votre site. Ensuite, vous devez fixer les objectifs de performance de votre site web. La recommandation pour le temps de chargement des pages en 2018 est d'être inférieur à 3 secondes. Selon les recherches de Google sur la vitesse moyenne des mobiles dans les différents secteurs d'activité, seule une fraction des sites web est proche de la vitesse recommandée.

Par conséquent, si vous optimisez la vitesse de votre site web à un niveau acceptable, vous obtiendrez un avantage significatif en termes de classement par rapport à vos concurrents.

Les méthodes d'optimisation de la vitesse du site web énumérées dans cet article sont très différentes et parfois la solution à un problème peut avoir un impact négatif sur un autre aspect de votre site. C'est pourquoi nous recommandons d'analyser la vitesse du site web après chaque changement afin de déterminer les actions qui donnent les meilleurs résultats.

Il existe plusieurs outils d'évaluation des performances qui valent la peine d'être essayés :

  • Google Pagespeed Insights est un outil gratuit de Google qui effectue un test de performance sur votre site et fournit des recommandations sur la manière d'améliorer les performances. Il fonctionne à la fois pour les versions desktop et mobile.
  • Lighhouse est un outil automatisé à code source ouvert qui permet d'améliorer la qualité des pages web. Vous pouvez l'exécuter sur n'importe quelle page web, qu'elle soit publique ou qu'elle nécessite une authentification. Il propose des audits pour les performances, l'accessibilité, les applications web progressives, le référencement, etc.
  • Pingdom est également un excellent outil pour tester la vitesse d'un site web, avec un certain nombre de fonctionnalités utiles. Il suit l'historique des performances de votre site web, émet des recommandations basées sur des données sur la manière d'améliorer la vitesse du site web et génère des rapports faciles à comprendre. Il propose des plans de surveillance gratuits et professionnels payants.
  • YSlow fournit également des recommandations sur la manière d'améliorer la performance de la page, établit des statistiques et résume tous les éléments.
  • Performance Budget Calculator est un outil gratuit qui permet de déterminer le type de contenu que vous pouvez utiliser pour que votre site fonctionne de manière optimale.
