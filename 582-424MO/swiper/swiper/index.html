Swiper

Swiper est une librarie JavaScript. Swiper vous permet de faire différents sliders avec une quantité d'options intéressantes comme la pagination, les sliders verticaux, la scrollbar, la lazy load, effet 3D, effet parallax...

Installation

Pour installer Swiper dans votre projet, il suffit d'ajouter un lien vers les fichiers CSS et JavaScript de Swiper.

Install Swiper

CDN : Content Delivery Network

On vous recommande vivement l'utilisation de fichiers CDN car ils sont largement mis en cache et se chargent très rapidement. Près de 2 160 000 de sites utilisent la librairie Swiper, donc si l'un de vos visiteurs a été sur un site qui a chargé Swiper à partir de ce même CDN, cela ne coûte aucune bande passante (chargement instantané !) Utilisation
  1. Ajouter les fichiers CSS et JavaScript de Swiper.
  2. Ajouter le HTML de mise en page de Swiper
  3. Initialiser l'instance de Swiper via JavaScript.

Ajouter les fichiers CSS et JavaScript de Swiper dans les pages HTML qui requiert l'utilisation de la librairie.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/swiper@9/swiper-bundle.min.css" /> <script src="https://cdn.jsdelivr.net/npm/swiper@9/swiper-bundle.min.js"></script> Lorsque vous ajoutez des librairies externes telles que Swiper ou Bootstrap, il est important de lier les fichiers js et css de ces librairies AVANT vos propres fichiers internes script.js et styles.css. L'ordre est important.

Ajouter le HTML de mise en page de Swiper.

<!-- Slider main container --> <div class="swiper"> <!-- Additional required wrapper --> <div class="swiper-wrapper"> <!-- Slides --> <div class="swiper-slide">Slide 1</div> <div class="swiper-slide">Slide 2</div> <div class="swiper-slide">Slide 3</div> ... </div> <!-- If we need pagination --> <div class="swiper-pagination"></div> <!-- If we need navigation buttons --> <div class="swiper-button-prev"></div> <div class="swiper-button-next"></div> <!-- If we need scrollbar --> <div class="swiper-scrollbar"></div> </div>

Initialiser l'instance de Swiper via JavaScript.

Finalement, vous devez initialiser Swiper en créant une instance de la classe Swiper et vous la logez dans une variable* JavaScript.

const mySwiperOne = new Swiper (swiperContainer, parameters); initialise le swiper avec 2 options:
  • swiperContainer qui est l’élement HTML ou le sélecteur CSS du conteneur du swiper. String|Obligatoire
  • Parameters qui sont les paramètres Swiper listés dans un objet. Object|Optionnel

De plus, lorsque vous initialisez une instance Swiper, vous pouvez changer les valeurs de plusieurs paramètres, à votre guise, afin de personnaliser les fonctionnalités du diaporama.

Voici l’exemple de base présenté par Swiper:

const swiper = new Swiper('.swiper', { // Optional parameters direction: 'vertical', loop: true, // If we need pagination pagination: { el: '.swiper-pagination', }, // Navigation arrows navigation: { nextEl: '.swiper-button-next', prevEl: '.swiper-button-prev', }, // And if we need scrollbar scrollbar: { el: '.swiper-scrollbar', }, }); Démos

Allons voir quelques démos : https://swiperjs.com/demos

Pour voir le code à utiliser, sélectionnez: Core

Paramètres de Swiper

À l’initialisation, on peut changer la valeur de certains paramètres prédéfinis. Il faut donc les lister dans un objet {} au format paramètre : valeur, tous séparés par une virgule.

const swiper = new Swiper('.swiper', { direction: 'vertical', grabCursor: true, effect: 'flip' }); Paramètres Swiper

Quelques exemples communs

  • direction: 'horizontal' | 'vertical'
  • effect: 'slide' | 'fade' | 'cube' | 'coverflow' | 'flip' | 'creative' | 'cards'
  • width: null | number
  • height: null | number
  • speed: number
  • breakpoints: object (donc une liste de propriété-valeur encadrée par etre deux {})

...parlant de breakpoint, et oui, les paramètres Swiper peuvent changer en fonction du breakpoint!

const swiper = new Swiper('.swiper', { // Default parameters slidesPerView: 1, spaceBetween: 10, // Responsive breakpoints breakpoints: { // when window width is >= 320px 320: { slidesPerView: 2, spaceBetween: 20 }, // when window width is >= 480px 480: { slidesPerView: 3, spaceBetween: 30 }, // when window width is >= 640px 640: { slidesPerView: 4, spaceBetween: 40 } } }) Modules Swiper et leurs paramètres

Un série modules Swiper sont à votre disposition pour personnaliser d'avantage votre Swiper. Ces modules comportent eux même leur propre paramètres. Voyons notamment:

  • navigation
  • pagination
  • scrollbar
  • autoplay
  • free mode
  • parallax
  • lazy loading
