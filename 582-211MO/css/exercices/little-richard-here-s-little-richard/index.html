Little Richard Requis L'album doit avoir une dimension de 500x500px et être parfaitement centré horizontalement dans la page. Une bordure blanche de 30px doit être présente tout le tour de l'album. (Attention cette bordure ne doit pas changer la dimension totale de l'album.) Un dégradé en diagonal ↗️ allant d'orange dans le coin inférieur gauche jusqu'à jaune dans le coin supérieur droit doit remplir le fond de l'album. La photo de M. Richard doit prendre 60% de la largeur de l'album, ne pas être déformée et être positionné dans le coin inférieur gauche ↙️. Le titre de l'album doit être entièrement dans une balise h1, avoir une taille de 60px et utiliser la police Anton. Le texte doit être inscrit en minuscule dans le HTML et une propriété CSS doit le transformer en majuscule. Ajuster l'espace verticale entre les lignes via la propriété line-height. Utilisez 3 balises span, une pour "Here's", "Little" et "Richard" afin d'ajuster leur couleur et les décaler. Ambiance Notes de cours 📚
