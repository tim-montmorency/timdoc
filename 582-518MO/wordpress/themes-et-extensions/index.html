 Thèmes et extensions

WordPress propose une interface simple et efficace, permettant facilement l'ajout de thèmes et d'extensions.

Pour ce faire, il faut premièrement se connecter au tableau de bord. Normalement, il suffit d'ajouter /wp-admin à l'url du site en question pour être redirigé vers l'admin et ensuite d'entrer ses informations de connexion: (nom d'utilisateur/mot de passe).

Thèmes

Il suffit dans la barre latérale de l'admin de sélectionner:

Apparence ➡️ Thèmes

Certains thèmes par défauts seront disponibles. Il est néanmoins possible d'en ajouter plus. Pour ce faire, cliquez sur le bouton Ajouter se trouvant à la droite du titre de la page.

Plusieurs thèmes réalisés par d'autres développeurs seront alors proposés. Si l'un d'entre eux correspond à vos besoins, il suffit de cliquer sur Installer et ensuite Activer pour qu'il devienne le thème courant.

Il est aussi possible d'ajouter un thème via un fichier .zip en appuyant sur téléverser un thème. Cette option est favorisée par les thèmes payants afin de restreindre l'accès au code, mais surtout, elle permet de téléverser un thème créé par soi-même!

Installons un thème de base à partir de cette dernière méthode afin de bien comprendre le fonctionnement de WordPress.

Extensions

Tout comme les thèmes, il suffit dans la barre latérale de l'admin de choisir Extensions (plugins) et de décider si nous souhaitons choisir parmi certaines extensions existantes ou si nous désirons en téléverser une.

Dans le cadre de ce cours, je vous invite à ajouter deux extensions existantes. Pour ce faire, cliquer dans la barre intitulée Rechercher des extensions:

  • Custom Post Type UI WebDevStudios
  • ACF | Advanced Custom Fields d'Elliot Condon
  • WP Multilang de Valentyn Riaboshtan
