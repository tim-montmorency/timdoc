 Types de contenu dans WordPress

Sans extension ajoutée, Wordpress fontionne sous une struture définie. Cette base vous permet de manipuler différents contenus, les enregistrer dans sa base de données et les afficher dans vos pages web. Voici les types de contenu inclus dans le coeur (core) de WordPress:

Types de contenu
  • Utilisateurs (Users)
  • Commentaires (Comments)
  • Type ce contenu (Post Type)
Types de post (post type)
  • Article (Post)
  • Page
  • Images ou fichier ajouté dans une page (Attachment)
  • Révision (Revision)
  • Menu de navigation
  • CSS ajouté (Custom CSS)
  • Révision de l'auto-sauvegarde (Changeset)

Vous pouvez consulter cet article pour les définitions.

Champs de base (article ou page)

  • ID
  • post_title
  • post_content
  • post_content_filtered
  • post_excerpt
  • post_author
  • post_date
  • post_date_gmt
  • comment_status
  • ping_status
  • post_password
  • post_name
  • to_ping
  • pinged
  • post_modified
  • post_modified_gmt
  • post_parent
  • guid
  • menu_order
  • post_type
  • post_mime_type
  • comment_count
