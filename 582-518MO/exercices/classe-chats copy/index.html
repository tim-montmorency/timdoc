Classe: Chats

Pour cet exercice vous nous allons utiliser les classes JavaScript afin de créer un générateur de chats à 9 vies 😸.

Médias 🖼️

Meow Vous n’avez qu’à écrire du JavaScript (Nul besoin de modifier le HTML ou le CSS). Requis Écoutez le clic sur le bouton "Créer un 😺". Chaque clic sur ce bouton doit appeler une classe JS insérant un nouvel élément contenant le texte "🐈" et la classe .chat. Chaque chat doit avoir une opacité de 100%. Cependant, chaque clic doit réduire son opacité de 10%. Autrement dit, si un chat a été cliqué 3x, il devrait avoir une opacité de 70%. Si un chat atteint une opacité de 0% (ou moins  🤷‍♂️), le chat doit être retiré de la page. Bonus Lorsqu'un chat est cliqué, déclenchez le son en pièce jointe Meow. Notes de cours 📚
