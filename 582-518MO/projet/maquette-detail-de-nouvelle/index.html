 Détail de la nouvelle-Designer

Cette étape concerne le designer de votre équipe. Vous devez créer la maquette de la page de détail de nouvelle de votre site. Autrement dit, une page de nouvelle typique.

Articles de référence

Voici les articles de référence devant être utilisée afin de créer votre maquette

  • Architecture sans frontière
  • Promis
  • Craque-Bitume
  • Les impatients
Requis La maquette doit être réalisée à partir du logiciel Adobe XD ou Figma et être cohérente avec le guide de styles et la maquette de la page d'accueil. La maquette doit respecter les requis clients concernant la page de détail de nouvelle. La maquette doit présenter la page de détail de nouvelle dans trois résolutions afin de montrer comment elle s'adapte sur:
- mobile (375px)
- tablette (768px)
- ordinateur classique (1200px) Ajoutez un style différent sur “nouvelles” dans la navigation principales afin d'indiquer à l'usager où il se trouve. Inclure un lien parmi le texte de la page de nouvelle afin de visualiser le style d'un lien dans un texte courant. S'il n'y en a pas dans le texte original, créez un lien sur un mot quelconque et dirigez le vers une autre page du site. Inclure aussi un italique et un gras. Exportez une version PDF de votre maquette. Remise

Poussez dans le dossier "sources" sur le répertoire GitHub de votre projet votre maquette XD ainsi que sa version PDF sous le nom "detail-de-nouvelle".
