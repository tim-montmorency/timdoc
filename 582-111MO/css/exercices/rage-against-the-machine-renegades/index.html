Rage Against the Machine - Renegades

Pour cet exercice, vous devez écrire du CSS afin de recréer les différentes pochettes de l’album Renegades du groupe Rage Against The Machine.

Ces pochettes sont un clin d’œil/parodie de l’œuvre LOVE de Robert Indiana.

Pour ce faire, vous devrez manipuler les formes d’un élément SVG à l’aide de variables CSS.

#ffffff #000000 #444444 Requis Créez-vous des variables CSS à partir des couleurs fournies. Tous les éléments se voyant attribuer une couleur en CSS devront utiliser ces variables. Le fond de la page doit être gris, tandis que l'album doit avoir une largeur équivalente à 80% du côté le plus petit de la page et être centré horizontalement ↔️ dans celle-ci. (Le SVG de la pochette attribuera automatiquement la hauteur désirée.) Par défaut, le fond de l'album doit être vert, les lettres doivent-être rouge et l'élément de contraste doit être noir. Lorsque l'album est survolé, inversez la couleur des lettres avec celle de l'élément de contraste. Lorsque la page atteint une largeur de 1000px ou plus, la couleur de fond de l'album doit devenir bleue. Le logo "Parental advisory" doit avoir une fond blanc ⚪️, une largeur correspondant à 15% de celle de l'album et être positionné dans le coin inférieur droit de celui-ci ↘️, à l'équivalent de 3.5% du côté le plus petit de la fenêtre. Lorsque la page est plus haute que large, le logo "Parental advisory" doit-être déplacé dans le coin supérieur droit de l'album ↗️, toujours à l'équivalent de 3.5% du côté le plus petit de la fenêtre.
