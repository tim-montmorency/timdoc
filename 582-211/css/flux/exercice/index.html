Exercice flux

Voici quelques idées pour pratiquer le flot (flux) en HTML/CSS de manière interactive. Ces exercices sont conçus pour vous aider à observer et manipuler le comportement des éléments en ligne, de bloc, et des propriétés qui modifient le flot.

Veuillez ne faire que les 2 premiers exercices. NE PAS FAIRE LES #3 ET #4. Exercice 1 : Inline vs Block

Objectif

Comprendre la différence entre les éléments inline et block.

Étapes

  1. Créez une page HTML avec le contenu suivant :
<!DOCTYPE html> <html lang="fr"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Exercice 1</title> <style> .block { background-color: lightblue; margin: 10px; } .inline { background-color: lightcoral; margin: 10px; } </style> </head> <body> <h2>Exemple d'éléments de bloc</h2> <div class="block">Bloc 1</div> <div class="block">Bloc 2</div> <div class="block">Bloc 3</div> <h2>Exemple d'éléments en ligne</h2> <span class="inline">Inline 1</span> <span class="inline">Inline 2</span> <span class="inline">Inline 3</span> </body> </html>
  1. Ouvrez la page dans votre navigateur.
  2. Essayez de changer les styles des classes .block et .inline :
    • Ajoutez une largeur fixe (width: 100px;).
    • Ajoutez une hauteur (height: 50px;).
    • Que remarquez-vous ?
Exercice 2 : Float

Objectif

Utiliser la propriété float pour placer des images et du texte.

Étapes

  1. Créez ce fichier HTML :
<!DOCTYPE html> <html lang="fr"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Exercice 2</title> <style> img { float: left; margin: 10px; width: 150px; } p { text-align: justify; } </style> </head> <body> <h2>Exemple avec Float</h2> <img src="https://via.placeholder.com/150" alt="Exemple d'image"> <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. NE PAS FAIRE LES #3 ET #4. Exercice 3 : Flexbox

Objectif

Découvrir comment Flexbox peut remplacer le flot normal.

Étapes

  1. Créez une page HTML avec le contenu suivant :
<!DOCTYPE html> <html lang="fr"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Exercice 3</title> <style> .container { display: flex; gap: 10px; border: 2px solid black; padding: 10px; } .item { background-color: lightgreen; padding: 20px; text-align: center; } </style> </head> <body> <h2>Exemple avec Flexbox</h2> <div class="container"> <div class="item">1</div> <div class="item">2</div> <div class="item">3</div> </div> </body> </html>
  1. Expérimentez :
    • Ajoutez flex-direction: column; pour empiler les éléments.
    • Ajoutez justify-content: center; ou space-between; pour ajuster l'espacement.
    • Changez les dimensions des .item pour observer leur comportement.
Exercice 4 : Grid

Objectif

Créer une grille simple pour organiser les éléments.

Étapes

  1. Essayez ce code :
<!DOCTYPE html> <html lang="fr"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Exercice 4</title> <style> .grid-container { display: grid; grid-template-columns: repeat(3, 1fr); gap: 10px; } .grid-item { background-color: lightyellow; padding: 20px; text-align: center; border: 1px solid black; } </style> </head> <body> <h2>Exemple avec Grid</h2> <div class="grid-container"> <div class="grid-item">1</div> <div class="grid-item">2</div> <div class="grid-item">3</div> <div class="grid-item">4</div> <div class="grid-item">5</div> <div class="grid-item">6</div> </div> </body> </html>
  1. Expérimentez :
    • Modifiez grid-template-columns pour ajouter ou supprimer des colonnes.
    • Utilisez grid-template-rows pour définir la hauteur des lignes.
    • Ajoutez un élément qui occupe plusieurs colonnes avec grid-column: span 2;.
