T02 - Publicité Web (15%)

Commande

Design graphique de 2 formats de publicité numérique pour une même campagne marketing :

  1. Bannière publicitaire « Annonces Display ». Plus précisément le format de bannière Pop-up.
  2. Bannière destinée aux réseaux sociaux, 3 formats à réaliser pour l’affichage sur :
    1. sur le fil d'Instagram
    2. sur le fil de Facebook
    3. dans les Stories

Les formats exacts des bannières vous seront donnés au prochain cours.

Instructions phase concept

SUJET

  • Choisir un produit culturel ou un événement culturel de votre choix afin de le promouvoir via une mini campagne de publicité Web auprès d'une audience que vous pourrez déterminer.
  • ou
  • Choisir une cause qui vous tient à cœur et cherchez à sensibiliser une audience que vous pourrez déterminer.

Lorsque choisi, entrez votre sujet ici:t02-publicite-sujets-choisis.xlsx

PLANCHES

  • Planche d'inspiration (moodboard) dans Figma - Cadre (Frame) format 1920x1080
  • Planche de styles - Cadre (Frame) format format 1920x1080
  • Les 2 planches dans le même fichier Figma.
  • Remplir le document analyse_structure-com.docx

REMISE PHASE CONCEPT

  1. Lien direct vers votre fichier Figma en ligne (Paramètres de partage: Anyone with the link can edit)
  2. Un dossier de remise nommé votrenomfamille_votreprenom_T01_582-201MO, ce dossier comprends:
    • Votre fichier travail Figma, exporté en fichier local en format .fig (save local copy - voir image ci-dessous)
    • Un sous dossier nommé exportations qui contiendra vos 3 cadres (frames) exportés en format .png
    • Un sous-dossier nommé typo qui contiendra la/les police(s) de caractères utilisés pour votre logo final (vous devez télécharger le fichier de la police, même si la police est nativement intégrée à Figma.
    • Un sous-dossier nommé références qui contiendra bvotre document d'anlayse rempli: analyse_structure-com.docx

Vous devrez ensuite compressé le dossier votrenomfamille_votreprenom_T02-CONCEPT_582-201MO en format .zip et le remettre dans devoir avant le prochain cours.

Dates importantes (notez à vos agenda!)

REMISES EN 3 PARTIES

  • 22 février (cours 5): Remise du concept
  • 29 février (cours 6): Remise finale (vous devez avoir terminé votre design) - Évaluation formative en classe
  • 14 mars (cours 7): Remise finale

Critères d'évaluation

PHASE CONCEPT (5%)

  • Positionnement des éléments adéquats
  • Palette de couleurs harmonieuses
  • Relation entre le langage plastique et le message juste (choix des images versus le sujet choisis)
  • Respect des consignes

PHASE FINALE (10%)

  • Palette de couleurs harmonieuses
  • Choix typographiques judicieux
  • Respect des critères d’esthétisme en Web design
  • Message, clientèle cible et contexte de présentation judicieux
  • Relation entre le langage plastique et le message juste
  • Respect des consignes
