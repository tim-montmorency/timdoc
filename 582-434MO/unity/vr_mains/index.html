Mains

Modifier l'apparence des mains dans Unity est très simple. Pour se faire, suivez les étapes suivantes pour chacun de vos controlleurs:

Sélectionnez le Left Controller ou Right Controller dans la hiérarchy. Dans la composante XR Controller, trouvez le paramètre Model Prefab qui est tout en bas. Glissez le modèle que vous voulez utiliser pour vos mains dans cette boîte. C'est tout! Mains personnalisées
