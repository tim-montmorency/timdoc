Pseudo-classes - Bouton

Pour cet exercice, vous devez utiliser les pseudo-classes afin de modifier l’apparence d’un bouton selon différentes conditions.

Aperçu du résultat 👇

Matériel Pen de départ

Couleurs 🎨

#90fcf9 #b1fdfb #856084 Requis Par défaut, la couleur de fond du bouton doit être turquoise. Au survole, la couleur de fond du bouton doit devenir turquoise pâle afin d'inciter l'usager à cliquer dessus. Lorsqu'il est cliqué, la propriété top du bouton doit avoir une valeur 15px afin de donner l'impression qu'il est enfoncé. Lorsqu'il reçoit le focus, le bouton doit avoir une bordure mauve de 5px.
