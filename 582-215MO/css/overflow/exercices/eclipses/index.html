Overflow - Éclipses

Pour cet exercice, vous devez compléter une mise en page illustrant différents stades d’une éclipse solaire.

Couleurs 🎨

#000000 Requis Faites en sorte que tous les textes soient centrés. Créez dans le code HTML 5 éléments correspondant aux lunes 🌙. Celles-ci devront avoir une dimension de 100x100px, être noires ⚫️ et être parfaitement rondes. Pour les éclipses de type contact, faites en sorte que seul ¼ de la lune soit visible 🌔. Pour celles de type partielle qu'elles soit au ¾ visible 🌓 et pour celle annulaire qu'elle soit entièrement visible et centrée 🌑. Attention que les excédents de lunes ne soient pas visibles.
